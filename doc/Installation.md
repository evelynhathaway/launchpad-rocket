###### From npm
```
$ npm install launchpad-rocket
```
###### As a dependency
```
# Create package.json
$ npm init

# Install Rocket and add to dependencies
$ npm install launchpad-rocket --save
```
###### From source
```
$ git clone https://github.com/evelynhathaway/launchpad-rocket
$ cd launchpad-rocket/
$ npm install
```
